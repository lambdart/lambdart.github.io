About

Well, everyone knows how hard writing can be (even more challenging using a non-native language), so this blog was created as an attempt to improve this particular skill, and as a positive side effect, add a little bit more information on the internet, the data sea that we all love and/or hate.

Although, the intent is to write about anything that sparkles interest, the main focus will be on the following subjects:

  • Philosophy

  • Functional Programming
  • Emacs
  • Lisp Dialects
